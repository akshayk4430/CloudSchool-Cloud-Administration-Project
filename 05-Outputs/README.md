
# 📊 Outputs – CloudSchool Provisioning Results

## 📌 Overview

This folder contains output files generated by PowerShell scripts during provisioning and validation.

These files are used to verify:

* user creation
* user updates
* skipped users
* script execution results

---

## 📄 Files

### 🔹 staff-provisioning-results.csv

Generated by:

```powershell id="zj7n0a"
03-create-staff.ps1
```

Contains:

* UserPrincipalName
* DisplayName
* Action
* Status
* Notes

### Action Values

* `Create` → user created
* `Update` → user updated
* `Skip` → no changes required

### Status Values

* `Success` → operation completed successfully
* `Failed` → operation failed

---

### 🔹 students-test-results.csv

Generated by:

```powershell id="7fz1rs"
02-create-students.ps1
```

Used for validating student provisioning logic.

---

## 🧪 Validation Approach

The provisioning scripts are validated using:

* controlled test runs
* comparison of expected vs actual results
* verification of attributes in Entra ID

---

## 📌 Key Purpose

* Provides visibility into script execution
* Helps identify errors or mismatches
* Confirms idempotent behavior (re-run results in skips)
